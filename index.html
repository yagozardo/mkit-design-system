Introduction

MKIT Design System

The Winsupply Marketing IT UX Design System is a shared library of design decisions, reusable components, and guidelines that help teams build consistent, accessible, and on-brand digital experiences.

Design Principles

  • Consistent — One visual language across all eCommerce surfaces
  • Accessible — WCAG 2.1 AA minimum on all components
  • Efficient — Reduce repeated decisions; focus on product problems
  • Scalable — Token-based so global changes propagate automatically
